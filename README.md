## Tech Stack

**Client:** Next.js

## Run Locally

Clone the project

```zsh
  git clone git@github.com:npomba/moscow.mba.git
```

Go to the project directory

```zsh
  cd moscow.mba
```

Install dependencies

```zsh
  npm i
```

Start the server

```zsh
  npm run dev
```
